# ROBO-GUESS 🤖

Juego de ahorcado estilo sci-fi con un robot que se daña según los errores.

## Stack
- React 18 + Vite
- Tailwind CSS
- Sin dependencias externas

## Desarrollo local

```bash
npm install
npm run dev
```

## Características
- 🤖 Robot animado con 7 estados de daño
- 🎮 Soporte para teclado físico
- 📊 Marcador de victorias/derrotas
- 💬 Mensajes del robot según el estado
- 🔊 Más de 50 palabras en español por categoría
- 📱 Responsive
